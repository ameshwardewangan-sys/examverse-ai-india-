# all_in_one
" One Platform. Every Exam. Every Skill. AI Powered Learning."

ExamVerse AI is a next-generation AI-powered Government Exam Preparation Platform for India. It provides UPSC, SSC, Railway, Banking, State PSC, Police, and other exam preparation with Previous Year Papers, Mock Tests, Current Affairs, Job Notifications, and Official Apply Links.

The platform also includes AI Tutor, AI Doubt Solver, Career Guidance, Weather & Exam Day Forecast, Gamified Learning System, and Owner/Admin Dashboard for full content management.

Supports Offline + Online hybrid learning with Firebase backend, AI features, and multi-device support (Web, Android, iOS via Flutter in future).

Goal: One platform for every exam, every skill, and every career path.
